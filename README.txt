Soyuz-u rocket launch simulation using operations on meshes

Like this launch, vaguely:
    http://www.youtube.com/watch?v=84XhArK-6q0#t=3m48s

Navigation Keys:           Action keys:
    w : forward                p : pause
    s : back                   o : toggle wireframe
    a : left                   f : toggle following the rocket
    d : right                  space : reset rocket and smoke
    q : up
    z : down

mouse:
    left button down, x motion : rotate about Y (up)
    left button down, y motion : rotate about X
    right button down, x or y motion : skewer about Z (in)
